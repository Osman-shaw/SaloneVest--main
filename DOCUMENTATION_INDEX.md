## üÜò TROUBLESHOOTING QUICK START

### "My app won't start"
```
1. Is backend running? ‚Üí Check npm run dev in backend folder
2. Is frontend running? ‚Üí Check npm run dev in frontend folder
3. Any error messages? ‚Üí Check COMPLETE_ERROR_FIX_GUIDE.md
4. Try clean install: npm install
```

### "Wallet won't connect"
```
1. Is Phantom installed? ‚Üí Check browser extensions
2. Are you on Solana devnet? ‚Üí Switch network in Phantom
3. Any errors? ‚Üí Check browser console (F12)
4. See: PHANTOM_DEBUG_GUIDE.md
```

### "Build errors"
```
1. Check error message in terminal
2. Look in: COMPLETE_ERROR_FIX_GUIDE.md or BUILD_ERROR_RESOLUTION.md
3. Try: npm cache clean --force && npm install
4. Restart development servers
```

### "API returns 404"
```
1. Is backend running? ‚Üí Check port 5000
2. Is endpoint correct? ‚Üí Check BACKEND_API_IMPLEMENTATION.md
3. Check: FIX_API_404_ERROR.md
4. Restart backend server
```
